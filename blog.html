Discuss the scope of var, let, and const

The scope of a variable refers to the part of the code in which the variable can be used. A var declaration can be globally scoped or function scoped. When a variable is declared outside a function using var , the variable is in global scope. That is, any variable declared outside a function block using the var keyword can be used in the entire window.
