# PH-Tube

PH-Tube is a simple website displaying fetched videos.

## Features

### 1. Video Display
- Fetches and displays videos.

## Usage

To use the website:
1. Clone the repository.
2. Open `index.html` in your preferred web browser.

## Technologies Used

- HTML
- CSS (Tailwind)
- JavaScript
